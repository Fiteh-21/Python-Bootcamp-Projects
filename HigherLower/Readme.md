# 📱 Higher or Lower: Instagram Edition

This repository contains a Python-based **command-line game** where players test their knowledge of social media influence by guessing which of two celebrities or brands has a higher follower count.

---

## 🎮 How to Play

1. **Compare A**  
   You are shown the **name**, **description**, and **country** of the first account.

2. **Against B**  
   You are shown the same details for a second, randomly selected account.

3. **The Guess**  
   Type **'A'** or **'B'** to guess who has more followers.

4. **Keep Going**  
   If you are correct, your score increases, and your previous **B** choice becomes the new **A** for the next round.

5. **Game Over**  
   The game ends as soon as you make an incorrect guess.

---

## 🛠️ Features

- **Dynamic Data**  
  Includes a dataset of 50 high-profile accounts, ranging from footballers like **Cristiano Ronaldo** to brands like **NASA**.

- **Recursive Gameplay**  
  The winner of the previous round carries over to the next **Compare A** slot, keeping the momentum going.

- **Score Tracking**  
  Provides real-time feedback on your current streak.

- **ASCII Art**  
  Custom logos for a classic terminal game feel.

## 🚀 Getting Started

### Prerequisites

- Python **3.x** installed on your machine.

### Installation & Execution

1. Clone the repository or download the `project14.py` file.  
2. Open your terminal or command prompt.  
3. Navigate to the directory containing the file:  

```bash
cd path/to/project14
```
4. Run the game:
```bash
python project14.py
```
