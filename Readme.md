# Python Bootcamp Projects

A curated collection of small Python projects and exercises completed during a Python bootcamp. Each folder contains a standalone program demonstrating fundamental Python concepts such as control flow, functions, file I/O, object-oriented programming, and simple GUIs.

## Contents

- BlackJack Capstone — Blackjack terminal game
- Brainy Quiz — Quiz app with question models
- CaesarShift — Caesar cipher implementation
- Calculator — Simple arithmetic CLI calculator
- Flash Cards — Language flash card app using CSV data
- Hangman — Classic Hangman game
- HigherLower — Simple comparison game
- mile to km converter — Unit converter example
- NumberGuessing — Number guessing game
- Password Generator — Random password generator
- Pomodoro — Pomodoro timer example
- PyPass Manager — Simple password manager using JSON
- Quizler — Quiz app (alternate)
- Roshambo Game — Rock-paper-scissors implementation
- Secret Auction System — Auction CLI app
- Snake Game — Basic Snake game
- Turtle Crossing Game — Turtle crossing arcade-style game
- Turtle Race — Simple turtle race using Turtle
- TurtleSketch — Drawing with Turtle
- Virtual Barista — Coffee machine simulator

Refer to each project's folder for detailed notes and individual READMEs.

## Requirements

- Python 3.8 or later
- Most projects rely only on the Python standard library. Some projects use `turtle` or other GUI modules included with standard Python distributions.

## Quick Start

To run a specific project, open a terminal, change into the project's directory, and run the main script. For example:

```bash
cd "BlackJack Capstone"
python blackjack.py
```

Or run another example:

```bash
cd "NumberGuessing"
python number_guessing.py
```

Check individual project folders for extra instructions, required assets (images, CSVs), or additional dependencies.
